## 简介
创建通用的用户权限系统,基于`springboot+springData+mysql+shiro`

## maven节点配置
```
<dependency>
  <groupId>org.jerry.light4j</groupId>
  <artifactId>member</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>
```
